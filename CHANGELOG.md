## v1.0.1 — 버그 수정 & 스마트 감지 강화

### 🔧 버그 수정

**스와이프 번역 덮어쓰기 수정**
스와이프(< >)로 다른 대사를 선택한 뒤 번역하면 이전 대사의 번역이 나오던 문제를 수정했습니다. 이제 스와이프 후 번역하면 현재 보이는 대사가 정상적으로 번역됩니다.

**수정창 캐시 꼬임 수정**
수정 모드에서 텍스트를 변경한 뒤 번역하면 변경 전 문장의 캐시가 나오던 문제를 수정했습니다. 텍스트를 수정하면 이전 캐시가 자동 초기화되어 새로 번역합니다.

**사전 단어 원문 병기 방지**
사전에 `Soap=소프`를 등록하면 번역 결과가 `소프[Soap]`처럼 원문이 함께 출력되던 현상을 수정했습니다. 이제 등록한 번역어만 깔끔하게 출력됩니다.

---

### ✨ 새 기능

**일본어 · 중국어 스마트 감지**
기존 한국어 ↔ 영어 자동 감지에 더해, 일본어와 중국어 텍스트도 자동으로 감지하여 설정한 목표 언어로 번역합니다.

---

### 💡 사전 활용 팁: 동일 단어 다중 등록

같은 단어에 여러 뜻을 등록할 수 있습니다! AI가 문맥을 읽고 적절한 번역을 자동 선택합니다.

```
Soap=소프
Soap=비누
```

→ "Soap handed the knife" → **소프**가 칼을 건넸다 (캐릭터명)
→ "wash with soap" → **비누**로 씻다 (물건)

**더 정확하게 하려면** 괄호로 힌트를 추가하세요:

```
Soap(character name)=소프
soap(item)=비누
```

이렇게 하면 AI가 대소문자와 힌트를 함께 참고하여 거의 100% 정확하게 구분합니다.
